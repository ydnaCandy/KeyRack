# UI 設計書 (UI/UX Design)

## 1. UI 全体方針・スタイル設計

- **デザインシステム**: Tailwind CSS (v3) を主軸とするユーティリティファーストCSS。
- **アイコン**: Lucide-React を採用し、モダンで統一感のあるフラットアイコンを使用。
- **アクセシビリティ**: 認識しやすいカラーパレット（ダークモードベース、もしくはOSに準拠したテーマ）の採用と、入力フォームにおけるフィードバックの徹底。
- **画面遷移機構**:
  - `App.tsx` 内の状態 (`isAuthenticated`, `hasMaster`) によって、認証画面とメインダッシュボード画面を動的に切り替える（React の条件付きレンダリング処理）。

## 2. 画面・コンポーネント構成

全体は大きく「認証フェーズ」と「機能・管理フェーズ（ダッシュボード）」に分かれている。

### 2.1. ルートコントローラー (`App.tsx`)
- アプリのライフサイクルと認証ステートを管轄。
- IPC 通信で `window.api.checkMaster()` を呼び出し、初回起動（登録フロー）かロック解除フローかを判断する。
- 認証が完了したら `Dashboard` へ遷移する。

### 2.2. 認証画面 (`components/Auth.tsx`)
マスターパスワードの入力・登録・ロック解除を行う画面。
- **初回起動時**: パスワードの設定フォーム（確認用と合わせて2回入力）。
- **起動時 / ロック画面**: パスワードの入力フォーム。
- **インタラクション**:
  - パスワード入力中はボタンをローディング状態にする。
  - エラー時（パスワードが違うなど）はわかりやすいエラーメッセージ（赤いテキストやダイアログ）を表示。

### 2.3. ダッシュボード画面 (`components/Dashboard.tsx`)
認証成功後にアクセスするメインUI。

- **サイドバー領域**:
  - 管理対象の切り替えメニュー群：「サービスログイン管理」「DB接続管理」。
  - 万が一の外部出力を担う「エクスポート」「インポート」ボタン。
  - アプリをロックするための「ロック」ボタン。
- **メイン表示領域**:
  - **検索バー**: 一覧から「名称」などで部分一致検索できるテキストフィールド。
  - **一覧リスト (カードまたはテーブルレイアウト)**:
    - 登録されたデータの一覧表示。
    - パスワード等のシークレットフィールドは、デフォルト「●●●●●●」のような非表示マスキング状態とする。
    - **パスワード表示/非表示トグル (目のアイコン)**: クリックで当該項目のパスワードのマスクを解除する。
    - **ワンクリックコピーボタン (クリップボードアイコン)**: アプリの外へ情報を持ち出す際に、平文でクリップボードへコピーさせる。
    - **編集 / 削除機能**: 各項目にある操作ボタン。

### 2.4. データ入力用モーダルコンポーネント

メイン画面の上にオーバーレイとして表示する追加・編集用のダイアログ画面。

#### `components/ServiceLoginModal.tsx`
- **対象**: サービスログイン情報の設定。
- **入力項目**: サービス名(必須)、URL、ユーザー名、パスワード(必須)、備考1、備考2。
- **バリデーション**: 必須項目が未入力の場合は保存ボタンを無効化（またはエラー表示）。

#### `components/DbConnectionModal.tsx`
- **対象**: DB接続情報の設定。
- **入力項目**: 名称(必須)、ホスト/DNS、IPアドレス、ポート、DB名、ユーザー名、パスワード(必須)。
- **バリデーション**: 名称・パスワードなどの必須チェックと、入力しやすいフォーマットの提供。

## 3. インタラクション・UXの向上策

- **パスワードコピー**: ユーザーがいちいち詳細画面を開かずとも、一覧画面からクリップボード連携でコピーできるよう操作の簡略化。
- **オフライン安心感の提示**: 常にローカルで動いている設計上、余分なローディング（不自然な遅延）を挟まず、サクサク動作する体験を提供する。
- **確認ダイアログの設置**:
  - エクスポート時: 「平文で出力されます」といった強力な警告表示（Windowsネイティブのモーダルまたは独自のオーバーレイダイアログ）。
  - 削除実行時: ご操作防止のため、必ず「削除してよろしいですか？」の確認を挟む。
