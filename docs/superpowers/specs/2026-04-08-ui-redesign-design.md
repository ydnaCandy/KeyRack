# KeyRack UI全面リニューアル 設計書

作成日: 2026-04-08

---

## 1. 概要

現行UIの「見た目・情報レイアウト・操作性」の全面改修。
ライトグレー基調のレトロゲーム風（スーパーファミコン風）デザインを採用し、
業務利用可能なフォントと遊び心のあるアクセントカラーを組み合わせる。

---

## 2. デザイン方針

### 2.1 カラーパレット

| 用途 | カラー | 値 |
|------|--------|----|
| 背景 | 白よりのグレー（薄ラベンダー） | `#f4f2f9` |
| カード・パネル | 白 | `#ffffff` |
| サブ背景 | 薄紫グレー | `#ede9f8` |
| ボーダー（通常） | 薄紫グレー | `#c9bce6` |
| ボーダー（強調） | 中紫グレー | `#b0a0d8` |
| テキスト（本文） | 濃い紫グレー | `#2a2440` |
| テキスト（補足） | 中間紫グレー | `#7a6a9e` |
| アクセント（主） | 紫 | `#7c3aed` |
| アクセント（副） | 明るい紫 | `#6d28d9` |

### 2.2 アクセントカラー（スーファミボタン風）

カードのバッジ・削除ボタンなどに使用し遊び心を表現する。

| 用途 | カラー | 値 |
|------|--------|----|
| Aボタン（赤） | 削除・警告 | `#c0392b` |
| Bボタン（黄） | 2番目のカードバッジ | `#d4a017` |
| Xボタン（青） | 3番目のカードバッジ | `#1a6fb5` |
| Yボタン（緑） | 4番目のカードバッジ | `#1a8a4a` |

### 2.3 タイポグラフィ（すべてSIL OFLライセンス・商用利用可）

| フォント | 用途 | ライセンス |
|----------|------|-----------|
| Silkscreen | ラベル・ボタン・タイトル・タブ | SIL OFL (Jason Kottke) |
| DotGothic16 | 本文・日本語テキスト・入力欄 | SIL OFL (FONTDASU) |
| IBM Plex Mono | パスワードマスク表示 | SIL OFL (IBM) |

フォントサイズ基準:
- 本文・入力欄: 16〜18px
- カードタイトル: 18〜20px
- ラベル（Silkscreen）: 8〜12px
- タブ（Silkscreen）: 12px

## 3. レイアウト設計

### 3.1 ダッシュボード

```
┌──────┬───────────────────────────────────────┐
│アイコン│ [🔍 検索バー................] [＋ 追加] │
│サイド │─────────────────────────────────────── │
│バー  │ ▶ SERVICE LOGINS — N件               │
│(62px)│ ┌─────────────────────────────────┐   │
│      │ │ [🌐] GitHub                      │   │
│ 🌐  │ │      user@example.com             │   │
│ 🗄️  │ │      •••••••••• 👁 📋   ✏️ 🗑  │   │
│      │ └─────────────────────────────────┘   │
│ ─── │ ┌─────────────────────────────────┐   │
│ 📤  │ │ [🌐] AWS Console                 │   │
│ 📥  │ │  ...                             │   │
│ ─── │ └─────────────────────────────────┘   │
│ 🔒  │                                        │
└──────┴───────────────────────────────────────┘
```

- サイドバー幅: 62px（アイコンのみ、ツールチップでラベル表示）
- ナビアイコン: サービスログイン / DB接続管理
- 下部アイコン: エクスポート / インポート / ロック（ロックは赤色）
- セパレーター線でグループ分け

### 3.2 カード

- 背景白・ボーダー付き・ホバーで浮き上がる（`translateY(-1px)`）
- 左端にカラーバッジ（40x40px、スーファミ色+3Dシャドウ）
- パスワードはデフォルトマスク、👁ボタンでトグル
- 削除ボタンは赤系（Aボタン色）で危険性を明示

### 3.3 認証画面

- ライトグレー背景に薄いグラデーション
- 白カード（ボーダー付き、紫グロー）
- ロゴアイコン: 紫グラデーション + 3Dシャドウ（`0 6px 0 #4c1d95`）
- タイトル: `KEYRACK`（Silkscreen、大きめ）

### 3.4 モーダル

- 背景オーバーレイ: 半透明紫がかった黒 + `backdrop-filter: blur`
- カード: 白・角丸14px・紫グロー影
- フィールド2列グリッド（ユーザー名・パスワード）
- ボタン: キャンセル（グレー）/ 保存（紫グラデーション3D）

## 4. インタラクション仕様

| 操作 | 挙動 |
|------|------|
| カードホバー | `translateY(-1px)` + ボーダー強調 |
| ボタン押下 | `translateY(2〜5px)` で3D押し込み感 |
| 検索入力フォーカス | 紫ボーダー + 薄紫グロー |
| パスワード表示トグル | 👁アイコンで当該カードのみ切替 |
| コピー完了 | 📋→✅ に一時変化（2秒） |
| 削除確認 | confirm()ダイアログ（既存の挙動を維持） |
